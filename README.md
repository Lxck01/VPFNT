# VPFNT – View Private Fortnite Tracker

A simple tool that allows users to **view the status of private Fortnite Tracker profiles**.

## 🔗 Live Tool
Use the tool here:  
https://lxck01.github.io/VPFNT/

## ✨ Features
- Check the **status of private Fortnite Tracker profiles**
- No login required
- Fast and easy to use

## 🚀 Usage
1. Open the website.
2. Enter the Fortnite username you want to check.
3. Click the check button.
4. View the profile status.
